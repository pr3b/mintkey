# dyte-mic-toggle

A button which toggles your microphone.

```html
<dyte-mic-toggle size="sm" class="dyte-el"></dyte-mic-toggle>
<dyte-mic-toggle size="lg" class="dyte-el"></dyte-mic-toggle>
<dyte-mic-toggle variant="horizontal" class="dyte-el"></dyte-mic-toggle>

<script>
  const elements = document.getElementsByClassName('dyte-el');
  for (const el of elements) {
    el.meeting = meeting;
  }
</script>
```
